What is Pyrocko?

A Python framework for geophysicists.


							import pyrocko

Software building blocks for seismological research.

  • Waveform handling
  • Forward modelling
  • InSAR surface displacement
  • Applications

Plugins to Snuffler

Snufflings

Compatible with ObsPy

Flavour existing ObsPy scripts with Pyrocko! 

							import obspy
							from pyrocko import obspy_compat
							obspy_compat.plant()

							# Read in MiniSEED data through ObsPy
							stream = obspy.read('test.mseed')

							# Start the Snuffler
							stream.snuffle()
