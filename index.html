Plugins to Snuffler

Called Snufflings

Compatible with ObsPy

Use your existing ObsPy scripts with Pyrocko! 

							import obspy
							from pyrocko import obspy_compat
							obspy_compat.plant()

							# Read in MiniSEED data through ObsPy
							stream = obspy.read('test.mseed')

							# Start the Snuffler
							stream.snuffle()
